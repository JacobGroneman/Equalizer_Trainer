# Equalizer Trainer 🔊 🎶
## version 1.0 | authored on April 27th, 2026

### Overview
This application trains sound engineers' ears in detecting frequencies for audio production using an equalizer.

### Modes
#### Tutorial
Learn this application's equalizer game through an interactive tutorial. Each step trains you in using the equalizer's various controls to guess the reference audio's frequencies.

#### Game
Play a fun minigame in guessing the audio frequencies of the reference tracks through using the equalizer's fader controls. Correct guesses lead to another round offering increasingly complex frequency levels. Can you hear all of them?

#### Sandbox
Use the equalizer with the built-in music player on sound files to enhance your listening experience. The playlist enables multi-track playback, with scrubbing & pause/play options for easy control.

### Installation
#### Windows x64
Locate & download the Windows x64 Edition .zip under "Releases" on Github.com. Unzip it & put the folder where you feel comfortable; press the application's .exe; enjoy the trainer.
